# references/ - Local Sources

Store local source files and manifests here. Each topic should have its own
directory:

```text
references/<topic>/
  SOURCES.md
  <source files or snapshots>
```

Every acquired source entry should include bibliographic data, DOI/arXiv/URL or
other access route, retrieval date, local file path, SHA256 hash, and relevant
theorem/equation/section locators.

Fetched sources are append-only. Do not edit a fetched paper or source
snapshot; add a corrected snapshot or manifest note instead.
