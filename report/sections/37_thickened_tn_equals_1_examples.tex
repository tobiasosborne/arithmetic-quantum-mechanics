\section{Thickened \texorpdfstring{\(t^n=1\)}{t to the n equals 1} Examples}
\label{sec:thickened-tn-equals-1-examples}

\subsection{Status and Dependencies}

This shard is the example layer for AQM-36.  It uses the factorization of
\(t^n-1\) from AQM-35 and the nilpotent-sensitive Weyl theorem of AQM-36.  No
new external source is used.

\subsection{Lamport Dependency Skeleton}

\[
\begin{array}{rcl}
D1&:&R_n=\mathbb F_3[t]/(t^n-1),\quad n=3^s m,\quad 3\nmid m.\\
L1&:&t^n-1=(t^m-1)^{3^s},\quad t^m-1\text{ squarefree}.\\
D2&:&A_\pi=\mathbb F_3[t]/(\pi^{3^s})\text{ for }\pi\mid t^m-1.\\
T1&:&\mathcal H_n^{\rm thick}=\bigotimes_{\pi\mid t^m-1}\ell^2(A_\pi),

      \ell^2(\mathbb F_3[t]/(\pi)),\quad
      \dim\mathcal H_n^{\rm red}=3^m.\\
E1&:&n=3\text{ gives one }27\text{-dimensional third-order jet qutrit}.\\

\end{array}
\]

\subsection{General \texorpdfstring{\(t^n=1\)}{t to the n equals 1} Thickening T1--T2}

AQM-35 proves that if \(n=3^s m\) with \(3\nmid m\), then
\[
  t^n-1=(t^m-1)^{3^s}
\]
and \(t^m-1\) is squarefree.  Let \(\mathcal P_m\) be the set of monic
irreducible factors of \(t^m-1\).  For each \(\pi\in\mathcal P_m\), define
\[
  A_\pi=\mathbb F_3[t]/(\pi^{3^s}),\qquad
  K_\pi=\mathbb F_3[t]/(\pi).
\]
The reduced layer of AQM-35 uses \(\ell^2(K_\pi)\).  The thickened layer of
AQM-36 uses \(\ell^2(A_\pi)\).

\paragraph{Theorem \(T1\).}
The nilpotent-sensitive Hilbert space is
\[
  \mathcal H_n^{\rm thick}
  =
  \bigotimes_{\pi\in\mathcal P_m}\ell^2(A_\pi),
\]
and
\[
  \dim_\mathbb C\mathcal H_n^{\rm thick}=3^n.
\]

\emph{Proof.}
Since \(|A_\pi|=3^{3^s\deg\pi}\),
\[
  \dim\mathcal H_n^{\rm thick}
  =
  \prod_{\pi\in\mathcal P_m}3^{3^s\deg\pi}
  =
  3^{3^s\sum_{\pi\in\mathcal P_m}\deg\pi}.
\]
Because \(t^m-1\) is squarefree and has degree \(m\), the sum of the degrees
of its irreducible factors is \(m\).  Therefore
\[
  \dim\mathcal H_n^{\rm thick}=3^{3^s m}=3^n.
\]

\paragraph{Theorem \(T2\).}
The reduced Hilbert space has dimension
\[
  \dim_\mathbb C\mathcal H_n^{\rm red}=3^m.
\]
Thus the thickened/reduced dimension ratio is
\[
  \frac{\dim\mathcal H_n^{\rm thick}}{\dim\mathcal H_n^{\rm red}}
  =
  3^{n-m}.
\]

\emph{Proof.}
AQM-35 gives
\[
  \mathcal H_n^{\rm red}
  =
  \bigotimes_{\pi\in\mathcal P_m}\ell^2(K_\pi),
\]
so
\[
  \dim\mathcal H_n^{\rm red}
  =
  \prod_{\pi\in\mathcal P_m}3^{\deg\pi}
  =
  3^m.
\]
The ratio follows from \(T1\).

\subsection{Example \texorpdfstring{\(n=3\)}{n=3}}

Let \(u=t-1\).  Then
\[
  R_3=\mathbb F_3[t]/(t^3-1)\cong\mathbb F_3[u]/(u^3).
\]
The reduced layer sees one closed point and one qutrit:
\[
  \mathcal H_3^{\rm red}\cong\mathbb C^3.
\]
The thickened layer is
\[
  \mathcal H_3^{\rm thick}=\ell^2(\mathbb F_3[u]/(u^3)),
  \qquad \dim\mathcal H_3^{\rm thick}=27.
\]
Write
\[
  a=a_0+a_1u+a_2u^2,\qquad
  \lambda(a)=a_2,\qquad
  \chi(a)=\exp(2\pi i a_2/3).
\]
For \(q,p\in R_3\),
\[
  T(q)|s\rangle=|s+q\rangle,\qquad
  R(p)|s\rangle=\chi(ps)|s\rangle,\qquad
  W(q,p)=T(q)R(p).
\]
If
\[
  q=\sum_{j=0}^2q_ju^j,\qquad p=\sum_{j=0}^2p_ju^j,
\]
then the commutator phase of \((q,p)\) with \((q',p')\) is
\[
  \exp\!\left(\frac{2\pi i}{3}
  \sum_{r+s=2}(p_rq'_s-p'_rq_s)\right).
\]
Thus the value coefficient pairs with the \(u^2\)-momentum coefficient, the
first jet pairs with the first-jet momentum coefficient, and the \(u^2\)
position coefficient pairs with the value momentum coefficient.

\subsection{Example \texorpdfstring{\(n=6\)}{n=6}}

Since \(6=3\cdot2\),
\[
  t^6-1=(t^2-1)^3=(t-1)^3(t+1)^3.
\]
There are two closed points, but each is third-order thickened.  Let
\[
  A_+=\mathbb F_3[t]/((t-1)^3),\qquad
  A_-=\mathbb F_3[t]/((t+1)^3).
\]
Then
\[
  \mathcal H_6^{\rm thick}
  =
  \ell^2(A_+)\otimes\ell^2(A_-)
  \cong\mathbb C^{27}\otimes\mathbb C^{27},
\]
so \(\dim\mathcal H_6^{\rm thick}=3^6\).  The reduced layer is
\[
  \mathcal H_6^{\rm red}\cong\mathbb C^3\otimes\mathbb C^3,
\]
with dimension \(3^2\).

\subsection{Conclusion}

For \(t^n=1\), the reduced layer depends only on the squarefree part
\(t^m-1\), while the thickened layer recovers the full coordinate-ring
cardinality.  The nilpotent variables are therefore genuine local jet modes,
not additional closed points.
