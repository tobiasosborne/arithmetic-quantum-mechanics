## Findings

- Severity: Major
  Location: `report/sections/41_cech_cohomology_quantum_descent.tex:136`
  Issue:
  The Cech homology boundary appears to reverse the covariant extension-map
  indices relative to the convention imported from AQM-26 and Curry.
  Evidence:
  AQM-41 defines, for \(s\in G(U_{i_0\cdots i_p})\),
  \[
    \partial s=\sum_j(-1)^j
    r_{U_{i_0\cdots i_p},\,U_{i_0\cdots\widehat{i_j}\cdots i_p}}(s).
  \]
  But AQM-26 defines a precosheaf extension as
  \(r_{U,V}:G(V)\to G(U)\) for \(V\subset U\)
  (`report/sections/26_presheaves_cosheaves_observable_nets.tex:84`).
  Curry's local source uses the same target/source order:
  \(r_{U_I^{(k)},U_I}(s_I)\), from the smaller full intersection to the
  larger omitted-intersection (`references/cosheaves/.../ch_abstract_sheaves.tex:277`).
  Suggested action:
  Swap the displayed arguments to
  \(r_{U_{i_0\cdots\widehat{i_j}\cdots i_p},\,U_{i_0\cdots i_p}}(s)\),
  or explicitly redefine the \(r\)-notation in this shard before using it.

- Severity: Major
  Location: `report/sections/41_cech_cohomology_quantum_descent.tex:143`
  Issue:
  The label Cech homology theorem ranges over all points \(x\in W\), but the
  current residue-qudit convention for affine finite-type schemes indexes
  finite Weyl sites by closed points only.
  Evidence:
  AQM-41 writes \(E(W)=\bigoplus_{x\in W}E_x\) and decomposes the proof over
  \(x\in U\) (`report/sections/41_cech_cohomology_quantum_descent.tex:143`,
  `:156`). `CONVENTIONS.md` (y) instead defines
  \(E(U)=\bigoplus_{x\in U\cap |X|_{\rm cl}}\kappa(x)^2\) and says non-closed
  points are not finite qudit sites. The statement is correct for finite
  spectra such as `Spec(k^n)`, where every point is closed, but is ill-typed if
  read across the broader residue-qudit layer after AQM-28.
  Suggested action:
  State the theorem over Weyl sites/closed points,
  \(x\in U\cap |X|_{\rm cl}\), or explicitly restrict T1 to finite spectra
  with all points closed.

- Severity: Minor
  Location: `report/sections/41_cech_cohomology_quantum_descent.tex:172`
  Issue:
  The proof of trivial higher label Cech homology uses a cone operator without
  handling simplices that already contain the cone vertex.
  Evidence:
  The proof says choose \(a\in I_x\) and set \(h(\sigma)=[a,\sigma]\). In the
  ordered Cech complex used here, chains are indexed by strictly increasing
  tuples (`report/sections/41_cech_cohomology_quantum_descent.tex:36`), so
  \([a,\sigma]\) is not a valid simplex when \(a\in\sigma\) unless a normalized
  degeneracy convention is added. The theorem is standard and likely correct,
  but the written contraction is incomplete.
  Suggested action:
  Define the contraction explicitly, e.g. \(h(\sigma)=0\) when \(a\in\sigma\)
  and otherwise insert \(a\) with the appropriate orientation sign, then check
  \(\partial h+h\partial=\operatorname{id}-\epsilon\).

- Severity: Minor
  Location: `report/sections/40_product_field_spectrum_qudit_stabilizers.tex:187`
  Issue:
  The product-field Weyl-basis proof uses nondegeneracy of the finite-field
  trace pairing but only says the additive character is nontrivial.
  Evidence:
  For \(q=p^r\), the trace in the \(z\neq z'\) case vanishes because a nonzero
  coefficient \(c\in k\) makes \(s\mapsto
  \operatorname{Tr}_{k/\mathbb F_p}(cs)\) a nonzero \(\mathbb F_p\)-linear
  functional. That is the separable trace-pairing nondegeneracy recorded in
  the local Stacks source manifest (`references/algebraic_geometry/SOURCES.md`,
  separable trace pairing lines 45278--45284), but AQM-40 does not cite it at
  the point where it is used.
  Suggested action:
  Add a one-sentence justification and local citation for trace-pairing
  nondegeneracy in the proof of Theorem T1.

## Cross-Shard Risks

AQM-41 is intended to synthesize AQM-26, AQM-27, and AQM-38--AQM-39. The
state/observable/stabilizer separation is mostly consistent with those shards,
but the label layer should consistently say whether it is over all topological
points or only closed Weyl sites. Otherwise later shards may accidentally
assign finite qudits to generic points.

AQM-40's main product-field conclusion looks consistent with convention (ad):
`Spec(k^n)` has \(n\) points and gives the ambient \(n\)-qudit system, while
stabilizer labels/phases are additional data. The only issue I found there is
the missing trace-pairing citation/proof detail.

## Suggested Actions

1. Fix the AQM-41 cosheaf boundary notation before downstream reviewers rely
   on its chain complex.
2. Tighten AQM-41 T1 to closed Weyl sites or finite all-closed spectra.
3. Make the cone contraction and AQM-40 trace-pairing step explicit.
