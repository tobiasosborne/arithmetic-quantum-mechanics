## Findings

- Severity: Major
  Location: `report/sections/39_two_qutrit_stabilizer_descent_counterexample.tex:155`
  Issue:
  The corrected statement asserts an equivalence
  `full global stabilizer state <=> global phased Lagrangian L <= E(U)`, but
  the cited proof does not establish the converse.
  Evidence:
  Lines 176-178 say the first equivalence is AQM-38 Theorem T1.  However,
  `report/sections/38_lagrangian_stabilizer_descent.tex:117` states only that
  a phased Lagrangian has a one-dimensional common `+1` eigenspace, and its
  proof through line 153 is one-way.  The broader converse is plausible and
  locally sourceable: `report/sections/08_symplectic_css_bridge.tex:203-206`
  says a stabilizer code becomes Lagrangian when `k=0`, and the Gross source
  manifest points to `references/symplectic_qecc/gross_0602001_source/poswig.tex:846-903`
  for stabilizer states from maximally isotropic subspaces.  But AQM-39
  currently cites only a theorem that proves `Lagrangian -> full state`.
  Suggested action:
  Either weaken the displayed statement to the proved direction, or add/cite a
  converse theorem saying that a full independent stabilizer state has
  Lagrangian projective label space, with the phase/eigenvalue data separated.

- Severity: Major
  Location: `report/sections/39_two_qutrit_stabilizer_descent_counterexample.tex:166`
  Issue:
  The phase part of the local-to-global statement is under-specified.  The
  shard says zero descent defect plus phase compatibility gives cover
  generation, but no phase descent condition is defined or proved.
  Evidence:
  AQM-38 proves a clean label statement for a fixed global phased stabilizer:
  `D_U(L; U_i)=0` iff the global labels are generated by local labels
  (`report/sections/38_lagrangian_stabilizer_descent.tex:181-208`).  It then
  says phase gluing is additional character compatibility on overlaps, but
  does not define that compatibility.  AQM-39 repeats this as part of the
  corrected theorem at lines 166-174.  For the disjoint two-qutrit example this
  is harmless, but for a genuine overlapping cover the asserted phased
  local-to-global statement is not yet a theorem in the shard.
  Suggested action:
  Scope T3 to the fixed-global-phase label theorem already proved, or add an
  explicit definition of local phase data, overlap compatibility, and the
  induced global character/eigenvalue condition.

- Severity: Minor
  Location: `report/sections/39_two_qutrit_stabilizer_descent_counterexample.tex:30`
  Issue:
  The Lamport skeleton says the state is entangled, but the proof only computes
  maximally mixed one-site marginals and does not state the entanglement
  criterion being used.
  Evidence:
  Lines 141-151 compute both one-site density matrices as `I_3/3`.  The theorem
  statement at lines 114-117 does not include entanglement, and the proof does
  not add the short step that a pure bipartite product state has rank-one
  one-site marginals, whereas `I_3/3` has rank 3.  AQM-27 uses Bell-state
  entanglement language elsewhere, but this shard should make its own local
  inference explicit if it advertises entanglement in the skeleton.
  Suggested action:
  Add a one-sentence derivation after the partial trace calculation, or remove
  `entangled` from the dependency skeleton and keep the claim to maximally
  mixed one-site marginals.

- Severity: Minor
  Location: `report/sections/39_two_qutrit_stabilizer_descent_counterexample.tex:149`
  Issue:
  The phrase "one-site restrictions are maximally mixed" is immediately tied
  to "no nonzero labels supported on a single site"; without a qualifier this
  can blur state restrictions, inherited stabilizer labels, and symmetries of
  the maximally mixed density matrix.
  Evidence:
  Lemma L2 proves `L cap E(A)=0` and `L cap E(B)=0` at lines 75-87.  Lines
  141-151 prove the reduced states are `I_3/3`.  But `I_3/3` commutes with all
  one-site operators as a density matrix, so "trivial one-site stabilizer
  restrictions" should be read as "no inherited nonzero +1 Weyl stabilizer
  labels from the global stabilizer," not as a statement about the commutant or
  all density-matrix symmetries of the marginal.
  Suggested action:
  Rephrase the conclusion around line 150 to say explicitly that the inherited
  one-site stabilizer label spaces are trivial, while the reduced states are
  maximally mixed.

## Cross-Shard Risks

AQM-38 and AQM-39 are correctly separating the state-presheaf marginal issue
from stabilizer-label descent, but AQM-39 line 188 calls nonzero defect the
"stabilizer-label analogue" of the AQM-27 state-presheaf phenomenon.  That is
reasonable as analogy, but downstream shards should not treat it as a Cech
cohomology or state-sheaf theorem unless phase descent and overlap data have
been formalized.

The convention entry `(ac)` in `CONVENTIONS.md` repeats the same broad
phrasing: "global Lagrangian stabilizer = full global stabilizer state at the
label level" and "global Lagrangian with zero descent defect = full global
stabilizer generated by the chosen local pieces."  If AQM-39 keeps the
equivalence language, the convention should point to a proved or sourced
two-way statement, not only the one-way projector calculation.

## Suggested Actions

1. Patch AQM-39 T3 so every displayed equivalence is either proved both ways
   or weakened to the direction established in AQM-38.
2. Add a compact phase-descent definition for overlapping covers, or explicitly
   defer phase gluing and state that AQM-38/AQM-39 prove only label generation
   for a fixed global phased stabilizer.
3. Tighten the two-qutrit wording: distinguish inherited one-site stabilizer
   labels from reduced-state symmetries, and either prove or omit the explicit
   entanglement claim.
